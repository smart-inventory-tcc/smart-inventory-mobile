# Smart Inventory UMKM

Repository ini berisi dua folder terpisah untuk pengembangan lokal:

- `backend/` : Express.js + SQLite backend untuk testing lokal.
- `frontend-mobile/` : Flutter mobile app untuk scan barcode dan integrasi dengan backend.

## Cara menggunakan

1. Jalankan backend terlebih dahulu.
2. Buka aplikasi mobile dan gunakan akun demo.
3. Scan barcode, lihat barang, dan lakukan transaksi masuk/keluar.
